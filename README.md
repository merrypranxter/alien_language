# Oraphine Script Library

This repository contains the specification and resources for **Oraphine**, an invented alien glyph language designed for art and prompt generation. The Oraphine script combines sacred geometry, biotechnology and signal aesthetics to create a consistent non‑human writing system. Glyphs are assembled from a limited set of structural parts—stems, chambers, wedges, crescents, forks—and accented with nodes, bars and halo marks. The resulting symbols are tall, narrow, slightly asymmetric and visually cohesive, suitable for relic engravings, temple markings, interface labels and more.

This repo acts as a style bible, a prompt library and a machine‑readable design system. The `docs/` folder contains human‑readable descriptions of how the language works, including construction rules, semantic behavior and layout modes. The `data/` folder holds JSON files that encode the glyph inventory, semantics, layout templates and prompt modules for programmatic use. The `examples/` folder shows sample inscriptions and use cases, and the `templates/` folder provides file skeletons for adding new content. See `REPO_MAP.md` for a detailed overview of the directory structure.

To begin, read `docs/00_overview/language_overview.md` and `docs/01_core_system/construction_units.md` to understand the building blocks of Oraphine. If you're expanding the system, follow the guidelines in `COPILOT_INSTRUCTIONS.md`.
