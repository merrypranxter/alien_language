# Glyph Card Template

Use this template to document a new Oraphine glyph.

- **ID:** e.g. `G025`
- **Name:** descriptive name such as *Left-Barbed Crescent Stem*
- **Class:** common, elevated, sigil, divider, number-like, modifier
- **Anchor:** stem, chamber or wedge
- **Components:** list of structural and accent parts used
- **Description:** concise visual description
- **Semantic Bias:** optional pseudo-meaning (e.g., action, vessel, sacred)
- **Variants:** optional list of known variant forms
