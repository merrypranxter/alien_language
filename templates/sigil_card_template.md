# Sigil Card Template

Use this template to describe a new sigil glyph.

- **ID:** e.g. `S030`
- **Name:** e.g. *Fourfold Crescent Seal*
- **Structure:** description of how crescents, chambers and nodes are arranged
- **Center:** the central element (chamber, node, eye)
- **Outer Rings:** description of rings or arcs
- **Accents:** any halos, nodes or bars used
- **Meaning:** optional pseudo-meaning or narrative association
