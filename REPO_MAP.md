# Repo Map

This repository defines an invented alien glyph language called Oraphine. It is organized into several top level areas to separate theory, data and examples.

- **docs/** – human‑readable design system. Each numbered subdirectory focuses on a different aspect of the script, from the core construction units to layout modes and prompting strategies.
- **data/** – machine‑readable JSON libraries that encode the glyph inventory, semantic roles, layout templates, prompt modules and visual DNA. These files feed creative tools and automation.
- **examples/** – sample inscriptions, prompt outputs and use case write‑ups demonstrating how to deploy the script in art and design.
- **templates/** – file skeletons to help contributors add new glyphs, sigils, prompts and layout descriptions with consistent formatting.
- **machine/** – operational instructions and schemas for tools such as Copilot or other repo‑driven generators.

When adding new files, please consult this map and follow existing conventions.
