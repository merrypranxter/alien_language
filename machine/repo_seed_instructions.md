# Repo Seed Instructions

These notes describe how to seed this repository. They are primarily for use by automated systems and should not be edited manually.

1. Create the directory structure as defined in `REPO_MAP.md`.
2. Populate `docs/` with descriptive markdown files detailing the theory and usage of the Oraphine script.
3. Populate `data/` with JSON files encoding the glyph inventory, semantics, layout templates and prompt modules.
4. Use the templates in `templates/` to add new glyphs, sigils, prompt modules and layout modes.
5. Expand the examples in `examples/` to demonstrate use cases.
6. Update `MANIFEST.json` and `REPO_MAP.md` whenever new sections are added.
