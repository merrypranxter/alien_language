# Semantic Categories

Group clusters loosely by thematic category to convey different tones:

- **Noun-like clusters:** more chambers and lower dots; fewer forks and wedges.
- **Verb-like clusters:** more forks and wedges; directional slashes.
- **Sacred clusters:** halos, mini crescents and crown wedges; more symmetry.
- **Warning clusters:** prominent wedges, slashes and sharp terminals; fewer enclosed forms.
- **Identity/name clusters:** an elevated glyph followed by several common glyphs.
- **Machine/interface clusters:** slashed chambers, double nodes, repeated divider marks and tight spacing.

Assigning visual behaviours to categories helps to use Oraphine consistently across contexts.
