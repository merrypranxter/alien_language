# Syntax Rhythm

Visual rhythm convinces viewers that Oraphine is real writing. Recommended repetition patterns include:

- Inserting a dotted glyph every three to five symbols.
- Placing a divider mark every four to eight glyphs.
- Introducing an elevated glyph every one to three lines of text.
- Repeating sacred markers at regular intervals.

These rhythms create predictable cadences that feel like grammar without the need for a strict language structure.
