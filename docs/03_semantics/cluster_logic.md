# Cluster Logic

Oraphine strings often appear in clusters of two to six glyphs. A typical cluster structure might include:

1. A simple opener (common glyph).
2. One or two core glyphs providing the main content.
3. An optional modifier‑heavy glyph that modifies the preceding ones.
4. An optional closing glyph or divider mark.

Longer ceremonial strings (seven to twelve glyphs) may include repeated patterns and elevated glyphs. Dividers appear every few glyphs to create readable rhythm. Vary cluster length and composition to add visual interest and suggest syntax.
