# Number-like Marks

Oraphine does not have a true numeric system, but certain marks resemble counting symbols and can imply quantity or indexing. These include:

- One node, two stacked nodes or three diagonal nodes.
- A dot inside a ring.
- A wedge paired with a node.
- A fork combined with a dot.
- A chamber repeated with multiple nodes.

Use these forms to suggest measurement, classification or enumeration within inscriptions or interface labels.
