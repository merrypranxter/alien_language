# Quick Reference Cheatsheet

This cheat sheet summarises key features of Oraphine.

- **Structural parts:** stems, chambers, wedges, crescents, forks.
- **Accents:** nodes, bars, halos.
- **Glyph classes:** common, elevated, sigil, divider, number‑like.
- **Deployment modes:** text bands, columns, circular seals, relic markings, interface labels, ruin fragments.
- **Stylistic modes:** inked, etched, luminous biotech, glitch, relic, screenprint safe.
- **Pseudo‑semantic logic:** chambers = nouns, forks = verbs, wedges = commands, crescents = modifiers.
- **Rhythm patterns:** dotted glyph every 3–5 symbols; divider every 4–8 symbols; elevated glyph each 1–3 lines.

Use this sheet as a handy reminder when designing new glyphs or prompts.
