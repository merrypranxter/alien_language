# Master Prompt Blocks

The **master prompt block** defines the core aesthetic of Oraphine for image generation models. Use it when you want the language itself to be the star. Example:

    invented alien written language, consistent glyph system, sacred geometric alien script, tall narrow glyphs built from repeating visual components: vertical stems, eye-shaped chambers, crescents, wedges, forked terminals, nodes, bars, and floating halo marks, asymmetrical near-symmetry, recurring syntax-like motifs, cohesive visual family, looks meaningful and structured rather than random, ceremonial yet functional, ancient-futuristic, mathematical and ritualistic, suitable for text bands, circular seals, vertical inscriptions, relic engravings, temple markings, and biotech interface labels, not based on any human alphabet

This block sets the stage for subsequent style and use‑case modules. Include it at the beginning of prompts to instruct models to generate a coherent Oraphine language.
