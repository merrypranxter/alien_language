# Compact Prompt Snippets

Compact snippets are shorter phrases you can mix into larger prompts when the script is not the focus but still present. Examples:

- **consistent sacred alien glyph language, structured not random**
- **cohesive ceremonial alien script built from repeating geometric-biotech forms**
- **real-looking invented alien writing, elegant, rule-based, uncanny**
- **sacred-tech glyph script with chambers, crescents, wedges, nodes, and halo marks**

These snippets convey the essence of Oraphine in a single clause.
