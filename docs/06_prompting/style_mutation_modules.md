# Style Mutation Modules

Add optional flavour to Oraphine by specifying stylistic mutations. Modules can be appended to the master block for different moods:

- **Temple‑Priest:** solemn, liturgical, high ceremonial order, hieratic priesthood energy.
- **Fungal‑Biotech:** organic yet geometric, spore-like nodes, membrane crescents, grown rather than written, living script.
- **Glitch‑Transmission:** signal-corrupted glyph language, doubled strokes, echo outlines, broken halos, partial redraw artefacts.
- **Crystal‑Engine:** faceted, sharp, precise, mineral intelligence writing, shard wedges, crystalline chambers, elegant machine-liturgical geometry.
- **Dream‑Oracle:** softly uncanny, luminous, celestial, oneiric, moonlike crescents, floating marks, psychic transmission feel.

Mix and match modules to experiment with different visual narratives while maintaining the core structure.
