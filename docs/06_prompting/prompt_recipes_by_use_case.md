# Prompt Recipes by Use Case

Different art applications call for tailored prompt recipes. Combine the master block with specific descriptors and constraints. Examples:

- **Temple Wall:** combine the master block with phrases like *ancient alien temple wall covered in a consistent sacred glyph language*, *weathered stone*, *chipped carvings*, *ritual geometry*.
- **Relic Object:** combine with *alien relic engraved with coherent inscriptions wrapping around the object*, *ancient-futuristic*, *strange and meaningful*.
- **Biotech Interface:** combine with *luminous biotech interface displaying a cohesive written glyph script*, *tightly organized short strings*, *circular clusters*, *signal-language aesthetic*.
- **Circular Seal:** combine with *circular alien seal composed of a central sacred sigil surrounded by rings of consistent written glyphs*, *asymmetrical near-symmetry*, *elegant and uncanny*.
- **Ruin Fragment:** combine with *fragment of alien inscription on broken stone*, *damaged but clearly rule-based script*, *weathered*, *enigmatic*.

Adjust the descriptive elements (material, environment, rendering) while keeping the script consistent.
