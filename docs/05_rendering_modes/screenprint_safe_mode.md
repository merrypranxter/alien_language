# Screenprint Safe Mode

Screenprint safe mode simplifies Oraphine for reliable printing on posters, T‑shirts or fabric. Guidelines:

- Use solid fills and avoid gradients or halftones.
- Keep linework at consistent medium weight.
- Eliminate very small details that may get lost during printing.
- Ensure adequate spacing between glyphs to prevent ink bleed.

This mode preserves the visual identity of Oraphine while ensuring legibility in print.
