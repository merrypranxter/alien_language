# Inked Mode

In inked mode, Oraphine is rendered as crisp black strokes on a light background. Use this style for print‑safe graphics, posters and clear diagrams. Features:

- Solid black lines and filled dots.
- No gradients, textures or glowing effects.
- Consistent line weight across the glyph.
- Clean negative space around symbols.

Inked mode emphasizes the graphic quality of Oraphine and is suitable for screen printing and vector design.
