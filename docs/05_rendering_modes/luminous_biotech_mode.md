# Luminous Biotech Mode

Luminous biotech mode portrays Oraphine as glowing or bioluminescent. Ideal for interface panels, biotech artifacts and mystical environments. Characteristics:

- Translucent chambers and crescents with internal glow.
- Nodes and halo marks appear as light sources.
- Soft gradients and subtle glows replace solid fills.
- Surrounding materials may appear organic or wet.

This mode blends organic and technological vibes, making the script feel alive.
