# Relic Mode

Relic mode renders Oraphine as if carved into bone or ancient organic materials. Characteristics:

- Uneven surfaces, cracks and weathering.
- Slightly irregular line widths and shaky strokes.
- Stains or discolorations in the material.
- Darkened recesses and highlighted ridges.

Use relic mode for ancient artifacts, bone carvings and sacred fossils.
