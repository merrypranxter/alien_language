# Glitch Mode

Glitch mode represents Oraphine as if corrupted or distorted by signal errors. Characteristics:

- Doubled or offset lines and echoes of strokes.
- Broken or fragmented halos and crescents.
- Partial redraw artefacts and random noise.
- Inconsistent alignment and repeated fragments.

Glitch mode is useful when representing damaged transmissions, corrupted interfaces or cosmic anomalies.
