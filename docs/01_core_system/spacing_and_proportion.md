# Spacing and Proportion

Oraphine glyphs occupy tall, narrow spaces. Keep these proportional guidelines in mind:

- **Height vs width:** most characters are taller than they are wide; aim for rectangles roughly twice as tall as they are wide.
- **Vertical stacking:** modifiers may sit above or below the main body, breaking the baseline.
- **Horizontal spacing:** leave irregular gaps between glyphs; the script should not read like evenly spaced Roman text.
- **Line rhythm:** when drawing bands of text, insert divider marks and larger glyphs to create visual pauses.
