# Interface Text

Interface text uses Oraphine in biotech dashboards and sacred machines. Characteristics:

- Short strings of 2–4 glyphs acting as labels or indicators.
- Lots of nodes, slashes and divider marks suggesting technical information.
- Circular clusters and seal‑like arrangements to signal controls or statuses.
- Clean strokes with luminous or translucent rendering in visual art.

Use interface text to suggest that Oraphine is actively used in machinery and digital systems.
