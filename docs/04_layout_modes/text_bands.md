# Text Bands

Text bands are horizontal strips of Oraphine glyphs. They are ideal for borders, posters, relic inscriptions and interface panels. Guidelines:

- Use mostly common glyphs with occasional elevated glyphs for emphasis.
- Insert divider marks at regular intervals to break long strings.
- Allow glyphs to float above or below the baseline; avoid rigid alignment.
- Vary spacing between clusters to create a lively rhythm.

Text bands can stretch across entire compositions or wrap around objects; treat them as ornamental yet structured elements.
