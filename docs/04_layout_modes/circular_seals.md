# Circular Seals

Circular seals arrange glyphs in rings around a central element. Use them for portal markers, interface medallions and relic seals. Guidelines:

- Place a sigil glyph or special mark in the center.
- Arrange common glyphs around the circumference, spacing them irregularly.
- Use crescents and wedges to radiate outward or inward.
- Embrace near symmetry but break it with offset halos, dots or split elements.

Circular seals should feel sacred and technical at the same time—a fusion of crop circles and machine UI.
