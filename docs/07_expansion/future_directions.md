# Future Directions

Ideas for expanding Oraphine:

- Develop dialects for different factions or timelines.
- Create numbering systems inspired by node patterns.
- Design ligatures that join glyphs together for cursive effects.
- Explore colour schemes and textures for different mediums (metal, stone, holographic).
- Build software tools that generate valid glyph combinations programmatically.

Document all new additions thoroughly to maintain coherence.
