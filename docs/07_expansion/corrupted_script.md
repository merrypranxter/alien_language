# Corrupted Script

The corrupted branch of Oraphine represents degraded transmissions or damaged inscriptions. Characteristics:

- Broken strokes, missing components and incomplete enclosures.
- Doubled or echoed lines creating phantom shapes.
- Reversed or mirrored glyphs indicating corruption.
- Unexpected slashes or static noise overlays.

Use corrupted Oraphine in glitch aesthetics, damaged relics or cosmic horror scenes.
