# Design Goals

The Oraphine script aims to:

- **Produce a cohesive non‑human writing system** that feels ancient and futuristic at once.
- **Restrict the visual vocabulary** to a handful of component shapes to enforce consistency.
- **Use asymmetrical near‑symmetry** to evoke alien design sensibilities.
- **Maintain tall and narrow proportions** with medium line weight.
- **Support multiple deployment modes** (horizontal bands, vertical columns, circular seals, relic engravings and interface labels) without changing the underlying glyph family.
