# Sample Inscriptions

Here are a few example Oraphine strings described in words. Use them as inspiration when drawing your own glyphs.

1. **Core core core divider core elevated core** – three simple common glyphs followed by a single dot divider, then another common glyph, an elevated glyph with a crown and halo, and a final common glyph.
2. **Core core elevated divider sigil** – two common glyphs, a stacked elevated glyph, a broken ring divider and a medium-sized sigil symbol.
3. **Sigil core core core** – a circular sigil at the start, followed by three common glyphs arranged radially around it.

When rendering these inscriptions, remember to vary spacing and insert divider marks to create rhythm.
