# Sample Prompt Outputs

These examples demonstrate how to combine Oraphine prompt modules for image generation:

- **Temple Wall Example:**  
  *invented alien written language...* + *ancient alien temple wall covered in a consistent sacred glyph language, weathered stone, chipped carvings, ritual geometry* + *avoid random decorative squiggles*  

- **Biotech Interface Example:**  
  *invented alien written language...* + *luminous biotech interface displaying a cohesive written glyph script, tightly organized short strings, circular clusters, signal-language aesthetic* + *organic yet geometric* + *signal-corrupted glyph language*  

- **Relic Fragment Example:**  
  *invented alien written language...* + *fragment of alien inscription on broken stone, damaged but clearly rule-based script, weathered, enigmatic* + *avoid human letters*  

Each recipe uses the master block, a use-case description and optional style modules and constraints.
